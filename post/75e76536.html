整数划分模型

发表于2022-06-23|更新于2023-10-30|算法
|阅读量:

题意

将n划分成多个数相加 有几种划分方案？

解法1 完全背包

f[i][j]表示从i个物品里面选恰好凑出体积为j的方案数

状态转移方程

f[i][j] = f[i - 1][j] + f[i][j - i]
写成一维: f[j] = f[j] + f[j - i]

Code

解法2 (k部分拆数)

f[i][j]表示由j个数组成i的方案数

状态转移方程

1. 有1 f[i - 1][j - 1] // 去掉1个1
2. 无1 f[i - j][j]     // 所有数减掉1
f[i][j] = f[i - 1][j - 1] + f[i - j][j]
